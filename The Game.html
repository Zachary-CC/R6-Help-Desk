THE GAME

The first thing you should do as soon as you make it to Rank 5 in the game’s progression system is to set up a custom multiplayer game and explore the maps. This is so you can get a feel for the type of terrain you’ll be exploring, which walls are destructible, and where the objectives might be.






Excluding the Recruit, Rainbow Six Siege features thirty operators at time of writing, split between attackers and defenders. As such, it can be a bit overwhelming knowing where to start.






There’s a variety of guns to choose from, including assault rifles, submachine guns, and shotguns. They all handle uniquely, deal different amounts of damage, and will react in a particular way once fired. Each operator has access to a particular set of weapons, and you can customize their design and attachments.
